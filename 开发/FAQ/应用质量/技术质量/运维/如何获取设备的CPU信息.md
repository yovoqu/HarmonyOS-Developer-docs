# 如何获取设备的CPU信息

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-performance-analysis-kit-22

可以通过以下命令来查看CPU信息：
 
```bash
// 查看CPU信息  
hdc shell param get const.product.cpu.abilist
```
