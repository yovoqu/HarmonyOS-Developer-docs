# Windows x86模拟器三方C/C++库使用限制

更新时间：2026-05-15 02:49:31

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-app-running-25

x86模拟器支持已x86化的C/C++三方库调试运行，未x86化的三方库开发者可以参考[tpc_c_cplusplus](https://gitcode.com/openharmony-sig/tpc_c_cplusplus)。
 
目前已经x86化的三方库如下：
  
| 三方库 | 备注 |
| --- | --- |
| MMKV | 自2.0.4-rc.4版本开始支持x86编译。 |
| ImageKnife | 自2.1.1版本开始支持x86编译。请重新安装imageknife再进行编译。 |
| ohos_videocompressor | 自1.0.3版本开始支持x86编译。 |
| ohos_coap | 自2.0.5版本开始支持x86编译。 |
| Unrar | 自2.0.2版本开始支持x86编译。 |
| commons-compress | 自2.0.2-rc.0版本开始支持x86编译。 |
| Smack | 自2.0.1-rc.0版本开始支持x86编译。 |
| ijkplayer | 自2.0.3-rc.0版本开始支持x86编译。 |
| ohos_mqtt | 自2.0.5版本开始支持x86编译。 |
| Mars | 自2.0.1-rc.1版本开始支持x86编译。 |
| mp4parser | 自2.0.2-rc.0版本开始支持x86编译。 |
