# 在DevEco Studio中提交代码时，如何自动格式化修改过的代码

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-coding-3

在Project目录中，选择要提交的文件或文件夹，右键点击，选择Git -> Commit File，打开Commit Changes窗口。在窗口中勾选Reformat code，即可自动格式化代码。
