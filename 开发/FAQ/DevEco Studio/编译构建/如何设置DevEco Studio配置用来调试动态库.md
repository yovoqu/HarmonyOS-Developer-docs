# 如何设置DevEco Studio配置用来调试动态库

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-compiling-and-building-53

在调试配置中选择 Debug type 为 Dual (ArkTS/JS + Native)、Native 或 Detect Automatically，在启动调试前完成配置，下一次调试时生效。
