# 编译报错“The service widget file contains one or more references to HSPs”

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-compiling-and-building-135

**错误描述**
 
服务卡片文件包含一个或多个HSP模块的引用。
 
**可能原因**
 
服务卡片文件中引用了HSP模块类型。
 

![](assets/编译报错“The%20service%20widget%20file%20contains%20one%20or%20more%20references%20to%20HSPs”/file-20260515130152728-0.png)

 
**解决措施**
 
在服务卡片文件中，移除关于HSP类型模块的引用。
