# 编译报错“module.json5 file not found.”

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-compiling-and-building-150

**错误描述**
 
未找到module.json5文件。
 
**可能原因**
 
模块下的module.json5文件缺失。
 

![](assets/编译报错“module.json5%20file%20not%20found.”/file-20260515130201608-0.png)

 
**解决措施**
 
在Stage模型项目中，确保每个模块包含module.json5文件。
