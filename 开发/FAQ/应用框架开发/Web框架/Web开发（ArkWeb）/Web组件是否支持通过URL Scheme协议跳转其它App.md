# Web组件是否支持通过URL Scheme协议跳转其它App

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkweb-72

Web组件支持通过URL Scheme跳转到其它App。开发者可以通过Web组件的onLoadIntercept回调拦截默认跳转逻辑，并在其中使用Deep Linking或App Linking的方式自定义跳转逻辑完成应用跳转。
