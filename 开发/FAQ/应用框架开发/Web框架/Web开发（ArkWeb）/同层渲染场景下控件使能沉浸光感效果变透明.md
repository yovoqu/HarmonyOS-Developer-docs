# 同层渲染场景下控件使能沉浸光感效果变透明

更新时间：2026-04-22 10:24:31

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkweb-100

**问题现象**
 
同层渲染场景下将ArkUI的控件设置沉浸光感效果时，发现控件背景变透明。
 
**解决措施**
 
API 23及以前SDK在同层渲染场景下暂不支持沉浸光感效果，建议在影响业务场景的对应控件上关闭沉浸光感或者关闭同层渲染。
