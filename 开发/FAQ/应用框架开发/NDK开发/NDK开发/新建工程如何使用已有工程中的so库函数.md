# 新建工程如何使用已有工程中的so库函数

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-ndk-39

可以将包含so库的工程编译成har包，然后在新工程中安装该har包，并通过import语句调用相应的so库函数。具体操作请参阅链接指南。
