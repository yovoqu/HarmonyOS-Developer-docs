# 如何快速生成class的setter和getter方法

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkts-101

1. 在类内部，右键点击，选择“生成”。
![](assets/如何快速生成class的setter和getter方法/file-20260515125648972-0.png)
2. 选择Getter and Setter。
![](assets/如何快速生成class的setter和getter方法/file-20260515125648972-1.png)
3. 选择要生成Getters和Setters的字段，点击OK，快速生成。
![](assets/如何快速生成class的setter和getter方法/file-20260515125648972-2.png)

![](assets/如何快速生成class的setter和getter方法/file-20260515125648972-3.png)
