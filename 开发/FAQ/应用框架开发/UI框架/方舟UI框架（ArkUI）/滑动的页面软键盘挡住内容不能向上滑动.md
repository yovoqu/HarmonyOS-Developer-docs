# 滑动的页面软键盘挡住内容不能向上滑动

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkui-105

计算软键盘的高度，然后将整体的margin-bottom设置为软键盘的高度。软键盘消失时，将margin-bottom设置为 0。软键盘高度可通过监听软键盘的显示事件获取。
