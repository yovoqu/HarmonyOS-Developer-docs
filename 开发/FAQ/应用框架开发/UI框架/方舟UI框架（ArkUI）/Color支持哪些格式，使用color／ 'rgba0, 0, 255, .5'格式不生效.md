# Color支持哪些格式，使用color: 'rgba(0, 0, 255, .5)'格式不生效

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkui-123

系统规格要求rgba格式必须完整书写，如rgba(0, 0, 255, 0.5)中透明度值前导0不可省略。
