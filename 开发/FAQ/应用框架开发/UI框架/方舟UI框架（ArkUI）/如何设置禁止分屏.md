# 如何设置禁止分屏

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkui-319

在应用开发阶段设置时，通过在module.json5配置文件中[abilities标签](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/module-configuration-file#abilities标签)下的supportWindowMode属性设置fullscreen字段，设置后应用将强制全屏运行，无法进入分屏模式。
