# 如何设置自定义弹窗位置

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkui-81

初始化自定义弹窗时，通过alignment参数设置弹窗对齐方式，通过offset参数设置弹窗偏移量。
