# 在容器组件嵌套的场景下，如何解决手势拖拽事件出现错乱的问题

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-arkui-172

PanGesture用于触发拖动手势事件。滑动的最小距离distance默认为5vp，设置distance值为1可提高灵敏度，防止事件错乱。
