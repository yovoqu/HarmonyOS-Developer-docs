# 如何获取resource目录下的资源

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-localization-4

可以使用以下两种方法获取资源文件：
 
1. 使用 \$r 或 \$rawfile 获取静态资源。程序运行时，资源路径不会改变。
 
2. 使用ResourceManager获取资源。适合动态获取，运行时可动态更改资源路径。
