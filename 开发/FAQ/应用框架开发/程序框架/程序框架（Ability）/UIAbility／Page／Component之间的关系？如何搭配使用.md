# UIAbility/Page/Component之间的关系？如何搭配使用

更新时间：2026-03-10 06:16:35

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-ability-66

UIAbility是应用中的组件，建议每个Hap包中只包含一个UIAbility。
 
一个UIAbility中可以包含一个或多个Page。开发者可以根据系统提供的能力实现页面间跳转。
 
一个Page中可以包含一个或多个Component。开发者通过不同的组件布局来实现不同的页面效果。
