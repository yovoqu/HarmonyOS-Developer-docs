# 如何监听文件或文件目录的变化

更新时间：2026-04-27 09:10:01

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-media-library-1

使用fileIo.createWatcher监听文件或目录的变化，并通过回调函数异步处理变化事件。应用需要根据业务需求控制通知处理策略，可以将变化信息存储到消息队列中，并定期刷新。
