# 接入资源包后台下载功能后如何使用指定测试设备进行自测

更新时间：2026-05-08 09:27:50

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/graphics-accelerate-assetdownload-faq-7

在AppGallery Connect正式发布资源包下载任务前，可以指定测试设备进行本地自测，具体操作请参见[测试下载功能](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/graphics-accelerate-assetdownload-release#测试下载功能)。在测试设备上测试下载功能符合预期效果后，再正式发布资源包下载任务。
