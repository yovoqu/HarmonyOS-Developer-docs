# AddConfig

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-addconfig

##### 函数功能

注册算子支持的AI处理器型号信息。
 
  

##### 函数原型

```text
void AddConfig(const char *soc);
```
 
  

##### 参数说明
 
| 参数 | 输入/输出 | 说明 |
| --- | --- | --- |
| soc | 输入 | 支持AI处理器型号。 |
 
 
  

##### 返回值

无
 
  

##### 约束说明

支持Kirin9020系列处理器
 
KirinX90系列处理器
