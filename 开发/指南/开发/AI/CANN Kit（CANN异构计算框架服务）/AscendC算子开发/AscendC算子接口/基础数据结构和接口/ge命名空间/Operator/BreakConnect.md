# BreakConnect

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-breakconnect

##### 函数功能

删除当前算子与前一个算子之间的所有连接关系，删除当前算子与下一个算子之间的所有连接关系。
 
  

##### 函数原型

```text
void BreakConnect() const;
```
