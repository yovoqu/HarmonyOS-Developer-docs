# Find

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-find

## 函数功能

查找子串在当前字符串中的位置。

## 函数原型


```text
size_t Find(const AscendString &ascend_string) const;
```


## 约束说明

无

## 参数说明


| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| ascend_string | 输入 | 要查找的子串。 |


## 返回值


| 类型 | 描述 |
| --- | --- |
| size_t | 子串的起始位置。 |
