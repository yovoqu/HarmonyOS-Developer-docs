# GetDim

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-shape-getdim

#### 函数功能

获取Shape第idx维的长度。
 
  

#### 函数原型

```text
int64_t GetDim(size_t idx) const;
```
 
  

#### 参数说明
 
| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| idx | 输入 | 维度索引，索引从0开始。 |
 
 
  

#### 返回值
 
| 类型 | 描述 |
| --- | --- |
| int64_t | 第idx维的长度。 |
 
 
  

#### 异常处理

无
 
  

#### 约束说明

无
