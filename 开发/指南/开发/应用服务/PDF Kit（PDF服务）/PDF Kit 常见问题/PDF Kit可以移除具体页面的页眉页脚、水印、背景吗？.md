# PDF Kit可以移除具体页面的页眉页脚、水印、背景吗？

更新时间：2026-03-09 02:50:43

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/pdf-faq-4

不可以，PDF Kit移除页眉页脚、水印或背景不会针对具体页面，请参考[页眉页脚](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/pdf-add-headerfooter)，[水印](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/pdf-add-watermark)或[背景图片](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/pdf-add-background)。
