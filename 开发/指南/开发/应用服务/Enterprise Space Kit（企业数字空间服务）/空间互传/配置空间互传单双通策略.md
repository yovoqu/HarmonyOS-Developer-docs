# 配置空间互传单双通策略

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/enterprisespace-policy-push

#### 场景介绍

Enterprise Space Kit支持HEM配置空间互传单双通策略。空间初始化时，通过调用接口配置空间互传应用文件外发策略。具体而言，单通表示只允许个人空间向企业空间发送文件，不允许企业空间向个人空间发送文件；双通表示允许个人空间和企业空间互相发送文件。



#### 接口说明

详细接口说明可参考[接口文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacedatatransfer#policypush)。

| 接口名 | 描述 |
| --- | --- |
| policyPush(policyContext: string): void | 配置空间互传单双通策略。 |




#### 开发步骤
1. 导入Enterprise Space Kit模块。

  
```text
import { fileTransfer } from '@kit.EnterpriseSpaceKit';
```

2. 调用[policyPush](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacedatatransfer#policypush)接口，配置空间互传单双通策略，并且查看打印信息。

  
```text

}
```
