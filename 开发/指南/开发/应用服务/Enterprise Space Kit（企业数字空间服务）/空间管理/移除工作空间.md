# 移除工作空间

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/enterprisespace-remove-workspace

## 场景介绍

Enterprise Space Kit为企业用户提供移除工作空间的能力。

## 接口说明

详细接口说明可参考[接口文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#removeworkspace)。
| 接口名 | 描述 |
| --- | --- |
| [removeWorkspace](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#removeworkspace)(localId: number): Promise | 移除工作空间。使用Promise异步回调。 |


## 开发步骤

导入Enterprise Space Kit模块。
```text
import { spaceManager } from '@kit.EnterpriseSpaceKit';
```

调用[removeWorkspace](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#removeworkspace)接口，移除工作空间，并且查看打印信息。
```text
const workspaceId: number = 100;
try {
  await spaceManager.removeWorkspace(workspaceId);
  console.info('Succeeded in removing workspace');
} catch (err) {
  console.error(`Failed to remove workspace. Code: ${err.code}, message: ${err.message}`);
}
```
