# 三方开发框架接入的问题

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/liveview-faq-2

实况窗随HarmonyOS整体框架支持三方平台，HarmonyOS为常见的三方开发框架提供了接入指南：
 
- [React Native](https://gitee.com/react-native-oh-library/usage-docs)
- [Flutter](https://gitcode.com/openharmony-tpc/flutter_samples/tree/br_3.7.12-ohos-1.1.0/ohos/docs)
- [Uni-app](https://uniapp.dcloud.net.cn/tutorial/harmony/intro.html)
