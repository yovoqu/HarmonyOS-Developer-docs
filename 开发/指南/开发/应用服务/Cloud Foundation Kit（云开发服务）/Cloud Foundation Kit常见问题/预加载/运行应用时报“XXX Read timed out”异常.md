# 运行应用时报“XXX Read timed out”异常

更新时间：2026-03-09 02:50:43

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cloudfoundation-faq-4

**问题现象**
 
运行应用时报“XXX Read timed out”异常。
 

![](assets/运行应用时报“XXX%20Read%20timed%20out”异常/file-20260514131850158-0.png)

 
**解决措施**
 
出现此错误，是因为云侧没有启动云函数实例，卸载应用重新安装即可。
