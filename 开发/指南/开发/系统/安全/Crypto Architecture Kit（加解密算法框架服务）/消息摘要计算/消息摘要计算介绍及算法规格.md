# 消息摘要计算介绍及算法规格

更新时间：2026-03-09 02:50:43

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/crypto-generate-message-digest-overview

消息摘要算法是一种能将任意长度的输入消息，通过特定运算生成固定长度摘要的算法。消息摘要算法也被称为哈希算法或单向散列算法。

## 支持的算法与规格

当创建MD消息摘要时，需要使用表中“支持种类”一列，指定MD消息摘要算法规格。
| 摘要算法 | 支持种类 | 字节长度 | API版本 |
| --- | --- | --- | --- |
| HASH | SHA1 | 20 | 9+ |
| HASH | SHA224 | 28 | 9+ |
| HASH | SHA256 | 32 | 9+ |
| HASH | SHA384 | 48 | 9+ |
| HASH | SHA512 | 64 | 9+ |
| HASH | MD5 | 16 | 9+ |
| HASH | SM3 | 32 | 10+ |
| HASH | SHA3-256 | 32 | 22+ |
| HASH | SHA3-384 | 48 | 22+ |
| HASH | SHA3-512 | 64 | 22+ |
