# @since

更新时间：2026-01-15 06:51:04

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-arktsdoc-since

@since标签表示在特定版本中添加了类、方法或其他符号。


## 语法

@since

## 示例

使用 @since：
```text
/**
 * Provides access to user information.
 * @since 1.0.1
 */
export function UserRecord(): void {}
```
