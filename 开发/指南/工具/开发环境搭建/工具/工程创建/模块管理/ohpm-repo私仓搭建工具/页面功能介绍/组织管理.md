# 组织管理

更新时间：2026-01-15 06:51:04

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpm-organization

在ohpm中包的命名格式为@&lt;group&gt;/&lt;package_name&gt;或者&lt;package_name&gt;。
 
其中group是组织，package_name是包名。当想要上传一个含有组织（例如@ohos/axios）的包时，在ohpm-repo中需要先创建出该组织（例如ohos）才能进行上传。
 

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fb/v3/j19x7ocvSyasrl7Ic0bJVg/caution_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=A66A309B8BB41A9C67DCAC0BEDDE133AF546A88B88253CE4C921B7819CA6A077)
 

在发布HAR/HSP包时，建议将组织名称包含在包名（package_name）中，便于管理和识别三方库。
 

 
同时在ohpm-repo中，只有组织成员才能上传该组织的包，如果一个包没有组织，那么后续版本更新只能由该包的第一任上传者上传。组织管理用于管理组织信息。
- 管理员用户的组织管理页面：可以查看所有组织的信息，编辑任何组织的管理员信息。此外，还能添加新组织、搜索现有组织，以及编辑和删除负责管理的组织。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ec/v3/xUdZsxlqR3KyXSqe72pnog/zh-cn_image_0000002571385834.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=06D0541C47FBD297B018AECB0AD7F05E403A8C8E49440277B120E8006D2964FD)


 
 
- 普通用户的组织管理页面：只能够看到当前用户所属组织的信息，能够查看和搜索组织，编辑和删除所管理的组织
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8/v3/ZF1SHo1-QuOSyy_8BYIetw/zh-cn_image_0000002602064943.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=4C3312B99B81A6E3EA4C4B0248B6CA6A6AB7C2FA406BC02C54E98CE9F1CDE7BB)


 

 1. 点击“新增”组织按钮，需要管理员用户权限，弹出添加新组织面板，可以新建一个组织，页面效果如下图所示：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/3f/v3/zp-j_37wRTeNYK0nM4Ajzg/zh-cn_image_0000002571385832.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=CD66DABA8504FF6EE2E5C5F51DAD030324254024D03ECABD9D556EEFE7ADC2CD)

2. 点击“详情”按钮，进入组织详情面板。包含描述、包、成员三个页签，其中描述页签是展示组织的基本信息；包页签展示该组织下仓库所上传的所有包信息；成员页签用于组织管理员对组织内成员的管理，可以输入用户名去添加成员或将成员移除出组织。
描述：展示组织的基本信息。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/da/v3/TtKTlhXoQvq0weaLpW3Mgg/zh-cn_image_0000002571385824.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=0770B28D96C93C602493B2B38CB51A77348A9DD36A5E0A7A0BAD3E3DCBF16D8B)

3. 包：展示该组织下仓库所上传的所有包信息。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/db/v3/IXpyUB8BR8SJnHzPOMHmUQ/zh-cn_image_0000002602064949.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=18E6FA7770516AA36079BADB9E12E08A56CCE714DE53B6FB12F091C4F61154C7)

4. 成员：组织管理员对组织内成员的管理，可以输入用户名去添加一位成员进入组织和将成员移除出组织；组织成员只有查看组织成员列表权限。
组织管理员页面：

5. 组织成员页面：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/36/v3/YJm6WpFpSU-NtMRaOv0iIw/zh-cn_image_0000002602064953.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=1B847C1B4DF57CCE92E36541E203E02D489082B4AAC83765FC174318E11B71DC)

6. 点击“新增成员”按钮：需要组织管理员权限，输入用户名，能够把该用户添加到组织中，成为组织成员。当组织成员添加后，成员用户将自动具有组织下所有包的维护者权限。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/81/v3/kpNcEMf5Sqy-uk2P6-zhlQ/zh-cn_image_0000002602064945.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=AECDCCC580B8FF797B43E416BC286F9F173D708B15412ABCAEA7CAA5DBB9EBF5)

7. 点击“删除”组织成员按钮：需要组织管理员权限。如果删除的成员是组织管理员，且没有其他组织管理员，则不允许删除，一个组织必须有至少一个组织管理员。当删除组织成员是普通成员时，如果当前用户对组织下的包具有维护者权限，权限将被删除；当删除的组织成员是组织管理员时，如果当前用户对组织下的包具有所有者权限，权限将被删除。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/af/v3/Dk67HmO-QjGVQ3Jx_O8vsQ/zh-cn_image_0000002571545462.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=250FBFF66734C45BCC5692D42A996F79FAE7E312C7FC7A53407BEF50BFCDE538)

1. 点击“编辑组织”按钮，需要组织管理员权限。进入编辑面板，可以修改组织的名称和描述，如果ohpm-repo内已经有该组织的包（组件数量不为0），则不允许修改名称：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e8/v3/hQ6SRB2YQTOn1VynoH7u9Q/zh-cn_image_0000002571385830.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=3AA1EB74FCEE319505F38452F8FD6248F3CC05AC45C22D35A7680AECC136C304)

2. 点击“编辑组织管理员”按钮，进入组织管理员详情页面，需要管理员用户权限。能够查看组织的管理员列表，并且对组织管理员进行新增或删除。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9e/v3/uIniNtvSSpm-pgnEXc5DJQ/zh-cn_image_0000002571385828.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=9F1B9C7ABA8675432BF892EAD887174BA23239BBDDDFF42E071CDC30E188E438)


  
点击“新增组织管理员”按钮，输入用户名将用户添加为组织管理员。当成功添加组织管理员后，当前用户将自动具有组织下所有包的所有者权限。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d5/v3/X1dB3rd6TuO_EOrWmNgJHQ/zh-cn_image_0000002571545460.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=CEF453FCAA685203EEF96A7F4F5F1E0ED0049637B7213320647413B3A2265838)

3. 点击"删除"组织管理员按钮，当组织管理员只有一个时，则不能被删除，一个组织必须有至少一个组织管理员。如果当前用户对组织下的包具有所有者权限，权限将被删除。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a4/v3/sIX59gmtTZekXLr-xIGJtA/zh-cn_image_0000002602184991.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=4A7E3E96B376210C97176ADE1AB5D4B71712FB61B346A93F2B2006C1945948ED)

4. 点击“删除”组织按钮，如果ohpm-repo内已经有该组织的包（组件数量不为0），则不允许删除组织。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/06/v3/rLTmTLc6SWu6XzYHzYQ-zQ/zh-cn_image_0000002602064955.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=6E9112FF7B9A9CA55447179B3CDC2A4943AB007ED82B0AFA421207E325516A92)

5. 点击搜索组织，组织搜索可以根据组织名称和组织管理员名称搜索。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/20/v3/AyneUdXbTiGp72Veztl0_Q/zh-cn_image_0000002602064951.png?HW-CC-KV=V1&HW-CC-Date=20260528T015020Z&HW-CC-Expire=86400&HW-CC-Sign=B3F8FDAFCFF5FB500FE39F759A9845A23F9D51277FC07D58A526C6D4D34AD45E)
