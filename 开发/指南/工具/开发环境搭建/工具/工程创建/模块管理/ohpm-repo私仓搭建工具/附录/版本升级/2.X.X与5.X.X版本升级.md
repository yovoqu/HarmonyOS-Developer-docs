# 2.X.X与5.X.X版本升级

更新时间：2026-01-15 06:51:04

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-upgrade-2xx_to_2xx

如需将ohpm-repo版本2.X.X/5.X.X版本升级到更高版本，可参考此文档。

1. 在升级之前，请务必备份好ohpm-repo私仓工具中的历史数据，避免因升级操作失误，导致数据丢失。备份的内容包括[](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-upgrade-110_to_2xx#li10435216234)ohpm-repo部署根目录内数据，db元数据以及store三方包数据，详细可参考[数据备份](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpm-repo-data-backup) 。

 2. 如果您想要改变db元数据和store三方包的存储方式，可在正确升级后参考[数据迁移文档指导](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpm-repo-data-migration)修改。


1. 多实例部署机器快速升级：在多实例部署中，其他机器同步骤1-6进行快速升级操作。
