# 用户管理

更新时间：2026-03-17 02:59:31

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpm-user-management

用户管理页面可以新增用户、修改用户类型、重置用户密码，删除用户和搜索用户，页面效果如下图所示：
 

![](assets/用户管理/file-20260514134328469-0.png)

 

 1. 点击“新增”按钮，弹出新增用户面板，输入新增用户的用户名和密码，新增用户首次登录将强制重置密码。填完用户信息后点击新增即可添加一个新用户，页面效果如下图所示：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d2/v3/WXRYShbeQ1S0bRV-dohNTQ/zh-cn_image_0000002602184963.png?HW-CC-KV=V1&HW-CC-Date=20260528T015021Z&HW-CC-Expire=86400&HW-CC-Sign=0EEFA3E15CA0C2E8B13353762FC4169A16398C41E4D9129C395D7E64F7FE4ACC)

2. 点击“编辑类型”按钮，弹出编辑用户类型面板，在此面板中可以修改用户类型成管理员或用户。点击确认修改用户类型，将出现密码输入框，由于管理员修改其他用户的类型是敏感操作，故需要输入当前操作账户的密码进行再次验证，页面效果如下图所示：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/44/v3/f_bvcVBjQdCfXIZQAEp1XQ/zh-cn_image_0000002571385796.png?HW-CC-KV=V1&HW-CC-Date=20260528T015021Z&HW-CC-Expire=86400&HW-CC-Sign=34495C769548097ED11B406620376F32299838F4FA3E990B5165286A7DF12A45)

3. 点击“重置密码”按钮，弹出重置用户密码面板，在此面板中可以通过点击生成新密码为用户生成随机新密码，并可通过点击复制图标将新密码复制进剪贴板（只有点击**确定**按钮才会重置密码）。点击确认重置密码，将出现密码输入框，由于管理员对其他用户重置密码是敏感操作，故需要输入当前操作账户的密码进行再次验证，页面效果如下图所示：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4d/v3/B0D54QTlRhCE_MjbumSKlw/zh-cn_image_0000002571545440.png?HW-CC-KV=V1&HW-CC-Date=20260528T015021Z&HW-CC-Expire=86400&HW-CC-Sign=9EEA789822373DE6CECAF65F24B8812A7982C8A3DCEB522D7986C82B27FDD60E)

4. 点击“删除”按钮，弹出删除提示，如果确定删除，需要点击按钮“是”，将出现密码输入框，由于管理员删除用户属于敏感操作，需要输入当前操作账户的密码进行再次验证，页面效果如下图所示：
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a1/v3/za7ydP6IQXe8nscs2kPjZQ/zh-cn_image_0000002571545436.png?HW-CC-KV=V1&HW-CC-Date=20260528T015021Z&HW-CC-Expire=86400&HW-CC-Sign=303D5E9A2708A02840C6C3DA164D3BF5377C26B99D0802B9842206CAF3F30D3F)


  当被删除的用户是某个三方包的唯一所有者时，禁止被删除。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/27/v3/vbw5ybA5QC-2kG7HqRqaFQ/zh-cn_image_0000002571385798.png?HW-CC-KV=V1&HW-CC-Date=20260528T015021Z&HW-CC-Expire=86400&HW-CC-Sign=3899D1750C510CA05BFCDAA12E9ED7BEF766FC8466EE4A5EF7E352C4DAFF3DAB)


5. 点击搜索框，支持指定用户类型（系统管理员/普通用户）和用户名模糊搜索，搜索页面效果如下图所示（以指定用户类型为系统管理员，用户名为admin为例）：
