# ohpm-repo --version

更新时间：2026-01-15 06:51:04

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpm-repo--version

查询ohpm-repo版本。


## 命令格式


```text
ohpm-repo --version
alias: -v
```


## 功能描述

打印ohpm-repo的版本号。

## 示例

执行以下命令，查看版本信息：
```text
ohpm-repo -v
```
