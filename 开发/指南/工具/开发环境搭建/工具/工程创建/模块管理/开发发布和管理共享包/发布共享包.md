# 发布共享包

更新时间：2026-01-15 06:51:04

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-har-publish

发布打包的HAR，可供其他开发者安装和引用。接下来将介绍如何发布HAR共享包。
 
> [!NOTE]
> OpenHarmony三方库中心仓仅支持HAR共享包发布，不支持HSP共享包发布。如需在应用内共享HSP，可将HSP共享包发布至私仓使用，请参考 ohpm私仓搭建工具 。

1. 在HAR模块中（与src文件夹同一级目录下），添加如下文件：

  
新建README.md文件：在README.md文件中必须包含包的介绍和引用方式，还可以根据包的内容添加更详细介绍。
2. 新建CHANGELOG.md文件：填写HAR的版本更新记录。
3. 添加LICENSE文件：LICENSE许可文件。
4. 重新[编译HAR模块](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-har#section7892044183814)，生成*.har文件。

  
> [!NOTE]
> 若修改了HAR包模块级oh-package.json5文件中version字段信息，请先执行Build > Clean Project指令，再重新进行Build全量构建。

5. 利用工具ssh-keygen生成公、私钥，可执行以下命令：

  
```text
ssh-keygen -m PEM -t RSA -b 4096 <span style="color: rgb(106,171,115);">-f ~/.ssh_ohpm/mykey </span>
```
 
> [!NOTE]
> ~/.ssh_ohpm/mykey 为私钥文件 mykey 的文件路径，按照实际情况指定。指定的私钥存储目录必须存在。 追加了.pub后缀的相应公钥文件会存放在和私钥相同的目录下。 OHPM包管理器只支持加密密钥认证，请在生成公私钥时输入密码。

6. 登录[OpenHarmony三方库中心仓](https://ohpm.openharmony.cn/#/cn/home)官网，单击主页右上角的**个人中心， **新增OHPM公钥，将公钥文件（mykey.pub）的内容粘贴到公钥输入框中。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fc/v3/ts_ttwTyTXeKjSFowcl5bQ/zh-cn_image_0000002571546454.png?HW-CC-KV=V1&HW-CC-Date=20260528T030655Z&HW-CC-Expire=86400&HW-CC-Sign=7565555F3D1B9930C6646B07DCD8DC486FD9C0BAB3CBAE45621250114316DDD1)

7. 打开命令行工具，将对应私钥文件路径配置到 .ohpmrc 文件中 key_path 字段上，可执行以下命令进行配置：

  
```text
ohpm config set key_path <span style="color: rgb(106,171,115);"> ~/.ssh_ohpm/mykey</span>
```

8. 登录[OpenHarmony三方库中心仓](https://ohpm.openharmony.cn)官网，单击主页右上角的**个人中心**，复制发布码，获取发布码并配置到 .ohpmrc 文件中，可执行如下命令：

  
```text
ohpm config set publish_id your_publish_id
```

9. 执行如下命令发布HAR，&lt;HAR路径&gt;需指定为.har文件的具体路径。

  
```text
ohpm publish <HAR路径>
```
