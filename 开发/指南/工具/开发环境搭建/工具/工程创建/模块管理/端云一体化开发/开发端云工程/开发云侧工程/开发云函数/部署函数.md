# 部署函数

更新时间：2026-01-15 06:51:04

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/agc-harmonyos-clouddev-deployfunc

完成函数代码开发后，您可将函数部署到AGC云端，支持单个部署和批量部署。
 
单个部署仅部署选中的函数，批量部署则会将整个“cloudfunctions”目录下的所有函数同时部署到AGC云端。
 
下文以部署单个函数“my-cloud-function”为例，介绍如何部署函数。
 1. 右击“my-cloud-function”函数目录，选择“Deploy 'my-cloud-function'”。
> [!NOTE]
> 如需批量部署多个函数，右击“cloudfunctions”目录，选择“Deploy Cloud Functions”即可部署该目录下所有函数。如“cloudfunctions”目录下同时存在云函数和云对象，云函数和云对象将会被一起部署到AGC云端。


  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/52/v3/JTiT9ftPQjyT19hHVnEr4g/zh-cn_image_0000002179498368.png?HW-CC-KV=V1&HW-CC-Date=20260528T015024Z&HW-CC-Expire=86400&HW-CC-Sign=3B1651EE8FA5D3FF016E25DDF7810D2DBDFAF9EF7763F2F2701A916311D9D518)

2. 您可在底部状态栏右侧查看函数打包与部署进度。请您耐心等待，直至出现“Deploy successfully”消息，表示当前函数已成功部署。

  
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/39/v3/OaAWmOmLSKObXuBppp8UZw/zh-cn_image_0000002179498360.png?HW-CC-KV=V1&HW-CC-Date=20260528T015024Z&HW-CC-Expire=86400&HW-CC-Sign=648AA552DE8CF86F691FBF50761284448956E50CF092A22C7A24FF1D5ACEA4F3)

3. 在菜单栏选择“Tools > CloudDev”。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/79/v3/dfCeLReuQHq-5HV3T5J9uA/zh-cn_image_0000002214858997.png?HW-CC-KV=V1&HW-CC-Date=20260528T015024Z&HW-CC-Expire=86400&HW-CC-Sign=203B321CDB15C8B6AA4D5987A7DBD5662F73BCB14144C9EBCBAA9CD1DA164FA4)

4. 在打开的CloudDev面板中，点击“Serverless > Cloud Functions”下的“Go to console”，进入当前项目的云函数服务页面。
![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/3f/v3/LJnscBs7RfuX4B7ow_09lw/zh-cn_image_0000002214704617.png?HW-CC-KV=V1&HW-CC-Date=20260528T015024Z&HW-CC-Expire=86400&HW-CC-Sign=027AC60341A680A62194A52FF44F25E84DEB5669EC27AB966E8F5D1392DF56F9)

5. 查看到“my-cloud-function”函数已成功部署至AGC云端，函数名称与本地工程的函数目录名相同。部署成功后，您便可以从端侧调用云函数了，具体请参见[在端侧调用云函数](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/agc-harmonyos-clouddev-invokecloudfunc)。
