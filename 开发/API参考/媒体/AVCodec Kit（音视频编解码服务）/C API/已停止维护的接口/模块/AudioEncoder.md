**支持设备：** Phone | PC/2in1 | Tablet | Wearable | TV

##### 概述

AudioEncoder模块提供用于音频编码的函数。
 
**系统能力：** SystemCapability.Multimedia.Media.AudioEncoder
 
**起始版本：** 9
 
**废弃版本：** 11
 
**替代建议：** 当前模块下的接口均已废弃，开发者可使用[AudioCodec](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-audiocodec)完成对应功能开发，单个接口的替代关系可查阅具体的接口说明。
