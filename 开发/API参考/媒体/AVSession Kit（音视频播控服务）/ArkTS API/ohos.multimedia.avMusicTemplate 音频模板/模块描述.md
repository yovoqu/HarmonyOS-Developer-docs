# 模块描述

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-avmusictemplate

音频模板提供数据写入和接收功能，可以实现媒体应用接入音频模板控制方进行统一风格页面展示，并响应音频模板控制方的操作指令。

该模块提供如下功能：

[AVMusicTemplate](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-avmusictemplate-avmusictemplate)：音频模板，可用于与音频模板控制方进行数据交互。


## 导入模块


```ts
import { avMusicTemplate } from '@kit.AVSessionKit';
```
