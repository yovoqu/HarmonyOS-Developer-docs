##### 概述

定义触摸测试信息项。触摸测试信息项包含触摸测试中子组件的信息，可以通过[OH_ArkUI_TouchTestInfoItem_GetXXX](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-ui-input-event-h#oh_arkui_touchtestinfoitem_getx)系列接口获取子组件信息。
