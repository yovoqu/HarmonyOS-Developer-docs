## 161001 监视渲染状态的节点数超过限制
**支持设备：** Phone / PC/2in1 / Tablet / Wearable / TV

**错误信息**

The count of nodes monitoring render state is over the limitation.

**错误描述**

当注册的监视渲染状态的节点数超过限制时，系统会产生此错误码。

**可能原因**

监视渲染状态的节点数超过限制。

**处理步骤**

请确保注册的监视渲染状态的节点数小于64。
