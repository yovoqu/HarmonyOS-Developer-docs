#### 概述

**支持设备：** Phone | PC/2in1 | Tablet | Wearable | TV

提供查询应用包信息的功能，包括应用包名、应用指纹、应用appId等。
 
**起始版本：** 9
 
  

#### 文件汇总

**支持设备：** Phone | PC/2in1 | Tablet | Wearable | TV
 
| 名称 | 描述 |
| --- | --- |
| ability_resource_info.h | 提供组件资源信息的接口，用于获取组件的以下信息：包名、模块名、组件名、图标、分身索引、是否为默认应用等。 |
| bundle_manager_common.h | 声明BundleManager定义的相关错误码。 |
| native_interface_bundle.h | 提供查询应用包信息的功能，包括应用包名、应用指纹、应用appId等。 |
