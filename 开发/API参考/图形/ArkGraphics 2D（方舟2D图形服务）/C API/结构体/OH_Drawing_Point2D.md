##### 概述

**支持设备：** Phone | PC/2in1 | Tablet | Wearable | TV

OH_Drawing_Point2D: 定义一个二维的坐标点。
 
OH_Drawing_Corner_Radii: 定义一个圆角半径，该圆角半径由x轴方向和y轴方向上的半径组成。
