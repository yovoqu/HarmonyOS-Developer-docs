# UsbDdk

更新时间：2026-04-20 06:34:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-usbddk
**支持设备：** PC/2in1

##### 概述

**支持设备：** PC/2in1

提供USB DDK接口，包括主机侧打开和关闭接口、管道同步异步读写通信、控制传输、中断传输等。
 
**系统能力：** SystemCapability.Driver.USB.Extension
 
**起始版本：** 10

**支持设备：** PC/2in1
