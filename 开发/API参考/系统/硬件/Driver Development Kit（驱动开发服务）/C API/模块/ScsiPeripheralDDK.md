# ScsiPeripheralDDK

更新时间：2026-04-28 03:31:56

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-scsiperipheralddk
**支持设备：** PC/2in1

#### 概述

**支持设备：** PC/2in1

SCSI Peripheral DDK是为开发者提供专门用于开发SCSI设备驱动程序的套件，提供了初始化DDK、释放DDK、打开关闭设备、读写设备等接口，并声明了SCSI Peripheral DDK API所需的宏、枚举变量和数据结构，用于在应用层进行SCSI设备驱动的开发。
 
**起始版本：** 18

**支持设备：** PC/2in1
 
| 名称 | 描述 |
