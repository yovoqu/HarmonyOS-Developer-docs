# BaseDdk

更新时间：2026-04-28 03:31:56

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-baseddk
**支持设备：** PC/2in1

##### 概述

提供Base DDK接口，包括创建共享内存、映射共享内存、取消映射共享内存以及销毁共享内存。
 
**起始版本：** 12
 
  

##### 文件汇总
 
| 名称 | 描述 |
| --- | --- |
| ddk_api.h | 声明主机侧访问的Base DDK接口。 |
| ddk_types.h | 提供基础DDK接口所使用的Base DDK类型，宏定义，枚举值和数据结构。 |
