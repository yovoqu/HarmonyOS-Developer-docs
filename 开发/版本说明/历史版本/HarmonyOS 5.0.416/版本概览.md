# 版本概览

更新时间：2026-01-21 11:07:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-releases/overview-504

HarmonyOS 5.0.4(16)在此前发布的HarmonyOS 5.0.3(15)的基础上，新增了Reader Kit以及少量接口，并对配套软件进行Release状态的发布。

详情可参见OS平台新增和增强特性和DevEco Studio新增和增强特性。


## 版本信息


> [!NOTE]
> 使用正确的配套关系进行应用开发可以获得更流畅的开发体验。
>  请在阅读版本更新和变更内容前，务必确认版本的配套关系是否与当前您所使用的开发套件是一致的。


### 5.0.4(16) Release开发者套件配套信息


| 软件包 | 发布类型 | 版本号 | Build Version | 发布时间 |
| --- | --- | --- | --- | --- |
| API版本 | Release | HarmonyOS 5.0.4 Release | 5.0.4(16) *注意：设备系统支持的API能力范围请以API版本为准。 | 2025/03/29 |
| DevEco Studio | Release | DevEco Studio 5.0.4 Release | 5.0.11.100 | 2025/03/29 |
| 5.0.11.110 | 2025/05/14 |  |  |  |
| SDK | Release | HarmonyOS 5.0.4 Release SDK | 基于OpenHarmony SDK Ohos_sdk_public 5.0.4.150(API 16 Release) | 2025/03/29 |


> [!NOTE]
> API版本请在设备的“设置”中点击设备名称，进入“关于本机”进行查询。当前系统版本配套的设备包括：Mate 60、Mate 60 Pro、Mate X5、MatePad Pro 13.2英寸、MatePad Pro 11英寸 2024款等。请升级至配套的版本进行应用开发。
>  不排除其他型号的设备通过其他途径升级至上述系统软件（ROM）版本，但设备所支持的开放能力请以“**关于本机**”页面显示的“**API版本**”为准。
>   DevEco Studio版本请从DevEco Studio界面菜单选择 > ”进行查询。请点击此处获取最新的DevEco Studio软件版本。SDK内置在DevEco Studio，安装DevEco Studio时自动安装配套版本SDK。具体版本请从DevEco Studio界面菜单选择 > ”进行查询。
