# 总览

更新时间：2026-01-21 11:07:33

来源：https://developer.huawei.com/consumer/cn/doc/harmonyos-releases/changelogs-overview-release

##### 概述

HarmonyOS NEXT每次新版本发布均会或多或少引入一些变更，这些变更可能会影响到您的应用。为了使开发者更清晰的感知变更对应用的影响，我们将变更做如下分类：
 
- **针对所有应用的变更**此类变更可能影响在HarmonyOS最新版本的镜像设备上运行的所有应用，无论您的应用是使用API 12或是更早的API版本进行开发。

  此类变更均需应用进行适配。
- **针对API 12应用的变更**此类变更仅影响您基于API 12的SDK进行开发和编译的应用。这类变更存在于以下两种场景：

  
API 11及以下版本的API在API 12发生变更，但变更仅影响使用API 12的SDK进行增量开发或重新编译。如应用已在API 11及以下版本完成编译，则不受变更影响。
- 在API 12的Beta版本中新增的API仍可能发生变更，此类变更可能影响基于API 12进行开发的应用。

 - **UX样式或效果的变更**仅对系统提供的UX样式和效果进行变更，开发者可以根据变更说明来判断是否影响应用整体风格或预期效果。
