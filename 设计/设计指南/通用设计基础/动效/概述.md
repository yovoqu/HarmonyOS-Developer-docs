#### 引力动效体系

HarmonyOS 动效引力体系，围绕回归本源设计理念，打造自然、流畅、品质一体的操作体验。

“引力”是自然中普遍存在的一种力，作用于万物并在其中表现出丰富多样的形态。我们将其提炼转化并运用于操作系统的动效设计中。转场动效表现出物体在运动过程中“力”的节奏感；手势动效打造出元素运动互相影响“力”的力量感；在特征动效中呈现出天体运动“力”的星体即视感。



| 力的即视感 将力赋予元素形象化、拟物化、动态化更直观地传递出设计理念，用在不同的场景上在表达新颖个性的同时又凸显着独特的品牌调性。 |  |
| 力的秩序感 元素有序的、一致的、节奏的运动能够创造更强的秩序感，让主要的信息更容易引起用户的注意力。在转场元素的运动编排上秩序感的体现尤为重要。 |  |
| 力的控制感 动效延伸着我们对于物理世界的认知，摩擦力、重力、斥力在交互界面中以另外的方式被感知着。操作界面时将力传递到界面上并与心理预期产生呼应，元素通过速度或节奏等显性因子，传达出仿真物理世界的感受。 |  |
